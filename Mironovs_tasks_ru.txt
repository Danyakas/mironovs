Интерактивность веб-страницы
Ожидаемые результаты
1. Назвать наиболее часто используемые события JavaScript.
2. Охарактеризовать значение событий во взаимодействии с пользователем.
3. Составить код на JavaScript для обработки событий.
4. Обосновать использование выбранных событий в своём проекте.
Информация, необходимая для выполнения заданий
1. JavaScript Events – https://www.w3schools.com/js/js_events.asp
2. HTML DOM Events – https://www.w3schools.com/js/js_htmldom_events.asp
Задания
1. Задание 1. Исследование.
Прочитай информацию о различных типах событий. В *.txt документе напиши:
1. как минимум 5 различных событий JavaScript (например: click, mouseover, mouseout, input, change);
2. короткое объяснение, когда используется каждое событие;
3. отметь 2 события, которые ты будешь использовать в своём проекте.
2. Задание 2. Дополнение веб-страницы.
Используя файлы, созданные на предыдущем занятии:
1. Убедись, что в HTML странице есть:
a. как минимум одна кнопка;
b. как минимум одно поле ввода (input, textarea или похожее).
2. В JavaScript (файл script.js):
a. привяжи как минимум 2 разных события, используя addEventListener();
b. для каждого события должно происходить видимое изменение на странице, например:
i. изменение текста;
ii. изменение стиля;
iii. показ уведомления;
iv. отображение/скрытие элемента.
Требование: события должны быть осмысленными (не только для демонстрации, а иметь значение для пользователя).
3. Задание 3. Анализ работы.
В *.txt документе напиши ответы на вопросы:
1. Какие события ты использовал?
2. Почему они подходят для твоего продукта?
3. Как эти события улучшают пользовательский опыт?
4. Задание 4. Сдача работы.
В e-klase (E-klase) отправь все *.txt файлы. GitHub
Сделай коммит в репозитории GitHub с описанием, например:
Add user interaction using JS events
Убедись, что:
● проект работает;
● учителю доступна ссылка на GitHub.
